\textbf{This is a problem box: } 
\begin{problembox}
Assume that the assumptions of the classical regression model apply but that the true value of the constant is zero, so that

\[
\bm{y} = \bm{X\beta} + \bm{u},
\]

where $\bm{X}$ does not contain a column of ones as a first column. Compare the expected value and variance of the least squares slope estimator computed without a constant term with that of the estimator computed with (an unnecessary) constant term, that is, using the model

\[
\bm{y} = \beta_0 + \bm{X\beta} + \bm{u}.
\]
\end{problembox}

\end{lstlisting}
